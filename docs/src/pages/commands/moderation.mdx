# Moderation Commands

- This is the list of our available moderation commands.
- These commands require appropriate permissions like `Manage Roles`, `Manage Channels`, etc. mentioned below in each command.

## addrole

<Callout>ns.addrole</Callout>

Assigns a specified role to a user.

**Usage**: `addrole <@user> <@role>`

- `<@user>`: The user to whom the role will be assigned.
- `<@role>`: The role to be assigned.

**Example**: `ns.addrole @nsgpriyansu @ns creator`

<Callout type="error">__Required Permissions:__ `Manage Roles`</Callout>

## createchannel

<Callout>ns.createchannel</Callout>

Creates a new text channel in the server with an optional description.

**Usage**: `createchannel <channel_name> [description]`

- `<channel_name>`: The name of the new channel.
- `[description]` (optional): A brief description of the purpose of the channel.

**Example**: `createchannel code-play-ground A place for creators thrive`

<Callout type="error">__Required Permissions:__ `Manage Channels`</Callout>

## createrole

<Callout>ns.createrole</Callout>

Creates a new role within the server.

**Usage**: `createrole <role_name> <color> [permissions]`

- `<role_name>`: The name of the new role to be created.
- `<color>`: The color of the new role to be created.
- `[permissions]`: The permissions of the new role to be created.

**Example**: `ns.createrole creators #ffffff SendMessages`

<Callout type="error">__Required Permissions:__ `Manage Roles` , `Manage Permissions`</Callout>

## deletechannel

<Callout>ns.deletechannel</Callout>

Deletes a specified channel from the server.

**Usage**: `deletechannel <#channel>`

- `<#channel>`: The channel to be deleted.

**Example**: `deletechannel #old-play-ground`

<Callout type="error">__Required Permissions:__ `Manage Channels`</Callout>

## deleteemoji

<Callout>ns.deleteemoji</Callout>

Removes an existing emoji from the server.

**Usage**: `deleteemoji <emoji>`

- `<emoji>`: The emoji to be removed.

**Example**: `ns.deleteemoji :unhappy:`

<Callout type="error">__Required Permissions:__ `Manage Emojis`</Callout>

## deleterole

<Callout>ns.deleterole</Callout>

Removes a specified role from the server.

**Usage**: `deleterole <role>`

- `<role>`: The role to be deleted.

**Example**: `ns.deleterole gg-devs`

<Callout type="error">__Required Permissions:__ `Manage Roles`</Callout>

## removerole

<Callout>ns.removerole</Callout>

Removes a specified role from a user.

**Usage**: `removerole <user> <role>`

- `<user>`: The user from whom the role will be removed.
- `<role>`: The role to be removed.

**Example**: `ns.removerole @xevy old-friend`

<Callout type="error">__Required Permissions:__ `Manage Roles`</Callout>

## updateemoji

<Callout>ns.updateemoji</Callout>

Updates the name of an existing emoji within the server.

**Usage**: `updateemoji <old_name> <new_name>`

- `<old_name>`: The current name of the emoji.
- `<new_name>`: The desired new name for the emoji.

**Example**: `ns.updateemoji :smile: :happy:`

<Callout type="error">__Required Permissions:__ `Manage Emojis`</Callout>
